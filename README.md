# She Can Foundation — Official Website

> A single-page website built for **She Can Foundation**, an NGO registered under the Indian Society Registration Act of 1860, dedicated to uplifting and empowering underprivileged women across India.


## 📌 About the Project

This project was built as part of a **Frontend Development Internship Task** for She Can Foundation. The goal was to create a clean, responsive, and visually appealing single-page website that represents the NGO's mission, focus areas, and values.

---

## ✨ Features

- 🌙 **Dark / Light Mode** — toggle button in the navbar
- 📱 **Fully Responsive** — works on mobile, tablet, and desktop
- 🎞️ **Scroll Animations** — elements animate in as you scroll
- 🖱️ **Custom Cursor** — smooth trailing cursor ring (desktop)
- 🎠 **Marquee Ticker** — scrolling banner with key focus areas
- 🃏 **Hover Interactions** — cards lift, buttons scale, nav links underline
- 📏 **Sticky Navbar** — shrinks on scroll, highlights active section
- 🖼️ **Embedded Logo** — base64 logo, no external image files needed

---

## 📄 Page Sections

| Section | Description |
|---|---|
| **Navbar** | Fixed navigation with logo, links, and dark mode toggle |
| **Hero** | NGO name, mission tagline, and call-to-action buttons |
| **Marquee** | Scrolling ticker with focus keywords |
| **About** | Mission statement and registration details |
| **What We Do** | Three focus areas — Education, Healthcare, Economic Empowerment |
| **Our Story** | Volunteer activity from She Can Foundation's LinkedIn |
| **Join Us** | CTA section linking to official website |
| **Footer** | Links, registration info, and social media |

## 🚀 How to Run Locally

No installation needed. Just:

1. Download or clone this repository
2. Open `index.html` in any browser

```bash
git clone https://github.com/your-username/she-can-foundation.git
cd she-can-foundation
# open index.html in your browser
```
